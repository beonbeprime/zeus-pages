# LP Builder - Crie sua Landing Page em minutos

Sistema inteligente que cria sua Landing Page de Low Ticket através de uma entrevista guiada.

---

## Como instalar

### Pré-requisito

Você precisa ter o Claude Code instalado. Se ainda não tem:

1. Acesse: https://claude.ai/code
2. Baixe e instale para o seu sistema operacional
3. Abra o Claude Code ao menos uma vez para configurar

---

### Instalação (1 passo)

Peça ao seu Claude para clonar o repositório:

```
Clone este repositório no meu computador: https://github.com/beonbeprime/zeus-pages
```

Ou, se você sabe usar o terminal:

```bash
git clone https://github.com/beonbeprime/zeus-pages lp-builder
cd lp-builder
```

---

## Como usar

1. Abra o Claude Code
2. Abra a pasta `lp-builder`
3. O sistema inicia automaticamente e faz as perguntas

Você escolhe entre:
- **Modo Rápido** (5 perguntas, 10 minutos)
- **Modo Extremo** (15 perguntas, 25 minutos, resultado mais personalizado)

---

## O que você recebe

- Arquivo `output/index.html` com sua LP completa
- Visual premium dark com as cores do seu estilo
- Copy com padrões de alta conversão para low ticket
- Link para visualizar ao vivo no browser

---

## Resultado

Sua LP terá:

- Hero com headline poderosa para o seu nicho
- Seção de dores do seu público
- Apresentação do seu método/produto
- Prova social e depoimentos
- Seção de oferta com preço
- Garantia e urgência
- Design responsivo (funciona no celular)
- Animações suaves de entrada

---

## Após a LP pronta

Para colocar no ar, você pode usar:

- **Netlify** (gratuito): arraste o arquivo `output/index.html` em netlify.com/drop
- **Hostinger** ou qualquer hospedagem: faça upload do arquivo
- **VPS própria**: informe ao Claude e ele faz o deploy

Lembre-se de substituir o link de compra `#comprar` pelo link real da sua Kiwify, Hotmart ou Monetizze.
